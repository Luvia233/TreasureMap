**版本：** V1.0

**项目类型：** 微信小程序

---

## 1. 项目概述

### 1.1 项目目标

开发一款微信小程序，帮助家庭用户记录和管理家中物品的放置位置，解决"东西放哪了"的常见问题。核心功能包括物品记录（支持手动和语音输入）、按空间/分类浏览、全文搜索（支持语音搜索）、家庭共享等。

### 1.2 目标用户

- **主要用户：** 个人或家庭成员
- **使用场景：** 记录物品位置、快速查找物品、管理家庭共享物品清单
- **设备要求：** 微信小程序（支持 iOS 和 Android）

### 1.3 核心价值

- 🔍 **快速定位** - 再也不担心找不到东西
- 👨‍👩‍👧 **家庭共享** - 全家人共享物品清单
- 🎤 **语音输入** - 说话就能记录，解放双手
- 📸 **照片辅助** - 一张照片胜过千言万语

## 5. 核心功能详细设计

### 5.1 用户认证流程

**步骤 1：微信授权登录**
- 用户点击"授权登录"按钮
- 微信 SDK 获取用户基本信息（昵称、头像）
- 后端生成用户记录，返回 session

**步骤 2：检查家庭状态**
- 如果用户已有家庭 → 进入首页
- 如果用户没有家庭 → 显示创建/加入家庭页面

### 5.2 家庭创建与加入

**创建家庭：**
1. 用户输入家庭名称
2. 后端创建家庭记录
3. 用户自动成为房主
4. 生成邀请码供家人加入

**加入家庭：**
1. 用户输入邀请码
2. 后端验证邀请码（检查有效性、有效期、剩余次数）
3. 验证通过后，用户加入家庭
4. 更新邀请码使用次数

**邀请码安全策略：**
- 格式：8-10位字母数字混合（如 `ABC123XY`）
- 有效期：7天
- 最大使用次数：5次
- 频率限制：同一IP每分钟最多尝试3次

### 5.3 物品添加流程

**方式 A：手动输入**
1. 用户点击"添加"按钮
2. 填写表单（物品名称、空间、容器、位置、分类）
3. 可选：添加照片
4. 点击"保存"提交

**方式 B：语音输入**
1. 用户点击"语音输入"按钮
2. 录音开始，显示录音动画
3. 用户说话，松开或超时后自动停止
4. 语音识别转为文字
5. 填充到"物品名称"字段
6. 用户补充其他信息后保存

**方式 C：拍照记录**
1. 用户点击"添加照片"按钮
2. 选择拍照或从相册选择
3. 上传照片到云存储
4. 显示照片预览
5. 用户补充物品信息

### 5.4 浏览与搜索功能

**首页浏览：**
- 按空间分组展示所有物品
- 每个空间卡片显示：
  - 空间图标和名称
  - 物品数量统计
  - 物品标签列表（显示前3个）
- 点击空间卡片展开查看该空间所有物品
- 点击物品进入详情页

**搜索功能：**
- **搜索范围：** 物品名称、空间、容器、位置、分类
- **搜索方式：**
  - 手动输入关键词
  - 语音输入（说即搜索）
- **结果展示：** 按分类分组显示
  - 分类标题 + 物品数量
  - 物品卡片显示位置信息
- **实时搜索：** 输入时显示建议

### 5.5 物品详情页

**显示信息：**
- 物品照片（大图展示）
- 物品名称
- 放置空间（带图标）
- 放置容器（带图标）
- 具体位置
- 物品分类
- 添加时间
- 添加人（显示昵称和头像）

**操作按钮：**
- 编辑：修改物品信息
- 删除：删除物品记录（需确认）

### 5.6 离线支持

**缓存策略：**
- 首次加载：获取家庭所有物品数据
- 本地存储：使用 wx.setStorageSync 保存数据
- 离线浏览：读取本地缓存数据展示
- 联网同步：切换到在线时自动拉取最新数据

**限制：**
- 离线时仅支持浏览和搜索
- 添加、编辑、删除操作需要联网

## 8. 安全性设计

### 8.1 用户认证安全
- 使用微信 OpenID 作为用户唯一标识
- 云开发自带登录凭证验证
- 用户数据按家庭隔离

### 8.2 邀请码安全
- 邀请码随机生成，长度 8-10 位
- 有效期限制（7天）
- 使用次数限制（5次）
- 频率限制（防暴力破解）
- 验证码机制（可选）

### 8.3 数据安全
- 家庭数据隔离（只能访问自己家庭的数据）
- 操作权限验证（房主 vs 成员）
- 敏感操作需二次确认

### 8.4 隐私保护
- 最小化获取用户信息（仅获取昵称和头像）
- 不存储敏感个人信息
- 数据加密传输

## 9. 错误处理

### 9.1 客户端错误处理

**表单验证错误：**
- 显示红色边框和错误提示
- 阻止提交
- 聚焦到第一个错误字段

**网络错误：**
- 显示 toast 提示
- 自动重试（最多3次）
- 提供手动重试按钮

**业务逻辑错误：**
- 显示友好提示信息
- 提供解决方案建议

### 9.2 服务端错误处理

| 错误码 | 说明 | 处理方式 |
|--------|------|---------|
| 1001 | 参数错误 | 显示参数错误提示 |
| 2001 | 未登录 | 跳转登录页 |
| 2002 | 无权限 | 显示无权限提示 |
| 3001 | 家庭不存在 | 提示检查邀请码 |
| 3002 | 邀请码无效 | 提示邀请码错误 |
| 3003 | 邀请码已过期 | 提示重新获取 |
| 3004 | 邀请码已用完 | 提示重新生成 |
| 4001 | 物品不存在 | 提示物品不存在 |
| 5001 | 服务器错误 | 显示服务器错误提示 |
